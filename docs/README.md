## 🎯 Cara Menggunakan Wiki

### Method 1: Baca di GitHub
1. Buka repository di GitHub
2. Klik tab **Wiki** (jika enabled)
3. Navigate menggunakan sidebar

### Method 2: Baca Local Files
1. Files tersimpan di `docs/wiki/`
2. Buka file `.md` dengan text editor atau Markdown viewer
3. Click links untuk navigate

### Method 3: Search
- **GitHub**: Gunakan search di GitHub wiki
- **Local**: Gunakan Ctrl+F dalam file
